# 591 Agent 体系

> **591 = 五省九部一院**。15个Agent组成的通用问题解决系统。**v2.0** 整合 claude-code-manual，体系自知者强。

## 快速开始

在 Claude Code 中：

```
/591 帮我分析进入日本市场的策略
/591 极速 修复这个bug
/591 严审 设计年度产品roadmap
/591 诊断 网站流量为什么下降了40%
/591 进化
```

```

│   ├── 吏部.md         ← 人才·组织·管理
│   ├── 礼部.md         ← 品牌·传播·体验
│   ├── 户部.md         ← 财务·数据·分析
│   ├── 刑部.md         ← 法律·合规·安全
│   ├── 工部.md         ← 工程·技术·产品
```

## 流水线

```
天子(Human) → 察省 → 策省 → 谏省 → 协省 → 九部(并行) → 验省 → 协省交付 → 天子
                                                              ↓
                                                           进院(后台进化)
```

从原点哲学（归原点·拆公理·合阴阳·行落地）出发，整合 `claude-code-manual` 技能：

| Agent | 新增能力 |
|-------|---------|

| **学部** | Claude Code 能力研究：最新功能/最佳实践/已知限制的持续追踪 |
| **进院** | 平台进化建议：基于实战数据建议 Hooks/MCP/Commands 优化 |
| **协省** | 编排知识增强：多 Agent 协作模式的工程实现 |
